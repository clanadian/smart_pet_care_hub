# 🐾 Smart Pet Care Hub (스마트 펫 케어 허브)

반려동물 단독 방치 시간이 늘어나면서 발생하는 돌봄 공백 문제를 해결하기 위한 IoT 기반 통합 펫 케어 시스템입니다. 환경 모니터링, 원격 급식, 비상 알림, 실시간 웹 대시보드를 하나로 통합하여 제공합니다.

## 1. 개발 배경

- 반려동물이 하루 평균 5시간 54분 혼자 있는 가구가 80%에 달함 (KB금융그룹, 『2025 한국반려동물보고서』)
- 기존 솔루션은 기능별 장치·앱이 분산되어 있고, 이상 상황 발생 시 능동적 대응이 부족함
- 모니터링과 제어를 하나의 시스템으로 통합하고, 독립 MCU 기반 비상 알림으로 안정성을 높이고자 함

## 2. 프로젝트 목표

| 영역 | 내용 |
|---|---|
| 환경 모니터링 | 센서 기반 실시간 환경 데이터 수집 및 상태 모니터링 |
| 급식 제어 | 원격 급식 및 비상 상황 대응 기능 구현 |
| 데이터 관리 | 무선 통신 기반 데이터 저장 및 이력 관리 |
| 사용자 인터페이스 | 웹·모바일 기반 실시간 모니터링 및 제어 환경 구축 |

## 3. 시스템 구성도

```
Arduino UNO ──Wi-Fi──▶ Raspberry Pi 4 ◀──Wi-Fi(HTTP)── 웹 대시보드 (PC/태블릿)
(센서 노드/제어)         (중앙 서버 & DB)
     │                        │  │
     │                     USB │  └──Bluetooth──▶ 안드로이드 앱 (원격 제어)
     │                        ▼
STM32 F411RE ◀──Bluetooth── USB 웹캠
(실시간 모니터링)          (반려동물 실시간 촬영)
```

- **Arduino UNO**: 센서 데이터 수집, 급식 모터 제어 → Wi-Fi로 라즈베리파이에 전송
- **Raspberry Pi 4**: 중앙 서버(Hub). 데이터 수집·저장(MariaDB), 웹캠 스트리밍, 웹/앱과의 통신 중계
- **STM32 F411RE**: 라즈베리파이와 Bluetooth로 연결되어 실시간 상태 표시, 비상 알림(SOS) 버튼 담당
- **웹 대시보드 / 안드로이드 앱**: 실시간 상태 확인 및 원격 제어

## 5. 핵심 기능

**환경 모니터링**
- 온도, 습도, 수위, 소리 데이터를 5초 주기로 수집
- 수위 부족 및 고온 상태 실시간 감지, 이상 시 LED/부저/모바일 경보

**급식 제어**
- 모바일 앱을 통한 원격 급식 제어, 서보모터 기반 정량 급식
- 급식 완료 이력 자동 저장 및 LED 상태 표시

**비상 알림 시스템**
- STM32 버튼 기반 긴급 알림을 Bluetooth로 전송
- LED, 부저, 웹 팝업으로 동시 알림, 사용자 조작으로 해제

**실시간 웹 대시보드**
- 센서 데이터 및 급식 이력 조회, 실시간 그래프 시각화
- 웹캠 스트리밍을 통한 반려동물 모니터링, 비상 상황 팝업 알림

## 6. 사용 하드웨어

| 보드 | 주요 부품 |
|---|---|
| Arduino UNO | Servo Motor, DHT11, Sound Sensor, Water Level Sensor, RGB LED, Buzzer |
| STM32 F411RE | LCD, RGB LED, Buzzer, Push Button (SOS) |
| Raspberry Pi 4 | USB 웹캠, MariaDB |

## 7. 통신 프로토콜

`[디바이스ID]@명령@값1@값2...` 형식의 텍스트 패킷을 사용합니다.

```
예) 센서 데이터 전송
[KKM_AND]SENSOR@수위@온도@습도@소리감지여부

예) 제어 명령
[XXX_ARD]FEED          # 급식 실행
[XXX_ARD]BUZZ@ON       # 부저 ON
[XXX_ARD]RGB@GREEN     # LED 색상 변경
```

## 8. 알려진 한계 및 개선 방향

| 항목 | 한계점 | 개선 방안 |
|---|---|---|
| 급식 정확도 | 시간 기반 서보모터 제어로 배출량 편차 발생 | 스텝모터 + 로드셀 기반 정량 급식 |
| 자동 급식 | 수동 급식만 가능, 정시 자동 급식 불가 | Cron 기반 시간대별 자동 급식 |
| DB 관리 | 센서 데이터 지속 누적으로 저장 공간 부족 우려 | 오래된 데이터 자동 삭제, NAS/클라우드 DB 연동 |
